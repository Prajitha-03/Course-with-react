# EduVance - E-Learning Platform

EduVance is a simple and interactive e-learning platform built with React. It allows users to browse courses, view chapters, and watch educational videos with real-time progress tracking.

## 🔧 Features

- Course and chapter listing sidebar
- Embedded video player
- Video progress tracking (% watched)
- Chapter descriptions
- Clean and responsive UI

## 🛠️ Tech Stack

- React
- HTML/CSS
- Vite
- Git & GitHub
